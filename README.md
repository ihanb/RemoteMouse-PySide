# RemoteMouse-PySide - 基于手势识别的远程鼠标控制系统

一个使用计算机摄像头进行手势识别，从而实现远程鼠标控制的Python应用程序。该系统利用MediaPipe进行手部检测，通过手势控制鼠标移动和点击操作。

## 功能特性

- **实时摄像头监控**：显示摄像头捕捉的画面
- **手势识别**：使用MediaPipe检测手部关键点
- **鼠标控制**：通过手势控制鼠标移动和点击
- **多摄像头支持**：可选择不同的摄像头设备
- **镜像模式**：支持画面镜像翻转
- **帧率显示**：实时显示摄像头帧率
- **用户友好的GUI界面**：基于PySide6的图形界面

## 手势说明

- **单指（食指）**：控制鼠标移动
- **双指（食指+中指）**：执行鼠标左键点击
- **手势识别状态指示**：界面显示当前手势识别状态

## 系统要求

- Python 3.7 或更高版本
- 支持摄像头的计算机

## 安装依赖

```bash
pip install -r requirements.txt
```

或者单独安装所需库：

```bash
pip install PySide6 opencv-python mediapipe pyautogui numpy
```

## 快速开始

1. 克隆或下载项目代码
2. 安装依赖库
3. 运行主程序：

```bash
cd src
python main.py
```

或者从项目根目录运行：

```bash
python -m src.main
```

## 使用步骤

1. **启动应用**：进入src目录后运行程序，将显示主界面
2. **选择摄像头**：从下拉菜单中选择要使用的摄像头
3. **打开摄像头**：点击"打开摄像头"按钮
4. **启用功能**：
   - 勾选"手势识别"启用手势检测
   - 勾选"鼠标控制"启用鼠标控制功能
   - 可选择是否启用"镜像模式"
5. **开始控制**：
   - 伸出右手食指来控制鼠标移动
   - 伸出右手食指和中指来执行左键点击

## 界面说明

- **摄像头选择**：下拉菜单选择摄像头设备
- **刷新摄像头**：扫描可用摄像头设备
- **开关摄像头**：打开/关闭摄像头
- **镜像模式**：画面水平翻转开关
- **手势识别**：启用/禁用手势检测
- **鼠标控制**：启用/禁用鼠标控制
- **视频显示**：实时显示摄像头画面和手势识别结果
- **状态信息**：显示当前摄像头和手势状态

## 技术栈

- **PySide6**：构建图形用户界面
- **OpenCV**：处理摄像头视频流
- **MediaPipe**：手部姿态估计和手势识别
- **PyAutoGUI**：执行鼠标和键盘操作
- **NumPy**：数值计算支持

## 配置选项

- **帧率**：默认设置为30 FPS
- **分辨率**：默认为640x480像素
- **检测置信度**：最小检测置信度为0.7
- **跟踪置信度**：最小跟踪置信度为0.5
- **最大手数**：最多检测2只手

## 故障排除

### 常见问题

1. **摄像头无法打开**
   - 确保摄像头工作正常
   - 检查是否有其他程序占用摄像头
   - 确认摄像头驱动已正确安装

2. **手势识别不准确**
   - 确保光线充足
   - 保持适当的手部与摄像头距离
   - 检查手部是否完全在摄像头视野内

3. **鼠标控制响应慢**
   - 降低摄像头分辨率
   - 减少环境中的杂光干扰
   - 确保计算机性能足够

### 权限问题

- 在某些操作系统上可能需要摄像头访问权限
- macOS可能需要额外的安全与隐私设置
- Windows可能需要以管理员身份运行
